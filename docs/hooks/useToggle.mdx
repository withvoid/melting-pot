# useToggle

```
const { on, onToggle } = useToggle();
```

## Examples

<Playground>
  {() => {
    const { on, onToggle } = useToggle();

    return (
      <div>
        <button onClick={onToggle}>{on ? 'on' : 'off'}</button>
      </div>
    );

}}

</Playground>

## API

useToggle() returns an Object

- on: [boolean] Status if on or off
- onToggle [function] Has function that toggles 'on' state.
