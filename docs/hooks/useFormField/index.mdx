# useFormField

```
const { value, set, reset, isEmpty, bind } = useFormField();
```

## Examples

<Playground>
  {() => {
    const form = {
      name: useFormField(),
      age: useFormField(),
      sex: useFormField(),
    };
    const [errorMessage, setErrorMessage] = React.useState('');

    const onSubmit = event => {
      event.preventDefault();
      const { areFieldsEmpty, emptyFields } = onValidateForm();
      if (areFieldsEmpty) {
        setErrorMessage(`Required fields ${emptyFields.join(', ')}`)
        return;
      }
      alert('Success ðŸ˜„');
      onReset();
    };

    const onReset = () => {
      // reset all input fields 
      Object.values(form).map(({ reset }) => {
        reset();
      });
      // clear error message, if any
      setErrorMessage('');
    };

    const onValidateForm = () => {
      const emptyFields = [];
      let areFieldsEmpty = false;
      Object.entries(form).map(([key, ({ isEmpty })]) => {
        if (isEmpty()) {
          areFieldsEmpty = true;
          emptyFields.push(key);
        }
      });
      return { areFieldsEmpty, emptyFields  };
    };

    const styles = {
      input: {
        display: 'block',
        boxSizing: 'border-box',
        width: '100%',
        height: 35,
        padding: 15,
        marginBottom: 15,
      },
      button: {
        border: 0,
        outline: 0,
        height: 35,
        width: 75,
        marginRight: 15,
        borderRadius: 2,
        backgroundColor: 'white',
        color: 'black',
        cursor: 'pointer',
      },
      error: {
        color: 'red',
      },
    }

    return (
      <div>
        {errorMessage !== '' && <p style={styles.error}>{errorMessage}</p>}
        <form onSubmit={onSubmit}>
          <input
            {...form.name.bind}
            placeholder="What is your name .."
            style={styles.input}
          />
          <input 
            {...form.age.bind}
            placeholder="What is your age .."
            style={styles.input}
          />
          <input 
            {...form.sex.bind}
            placeholder="What is your age .."
            style={styles.input}
          />
          <button type="submit" style={styles.button}>Submit</button>
          <button type="button" onClick={onReset} style={styles.button}>Reset</button>
        </form>
      </div>
    )
  }}
</Playground>

## API

useFormField(initialValue) returns an Object

* value: [String|Number] The value of input
* set: [Function] A setter method to set value of input
* reset: [Function] Reset form value to initialValue
* isEmpty: [Function] Returns a boolean if field is empty or not
* bind [Object] Has `value` & `onChange` in object which we can spread in the form
